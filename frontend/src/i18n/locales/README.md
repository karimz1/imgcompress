# Translation Notice

Some translations are online-tool assisted and may be imperfect.

If a translation does not look right, community improvements are highly appreciated. Please improve the relevant locale file here:

https://github.com/karimz1/imgcompress/tree/main/frontend/src/i18n/locales

Together we can make ImgCompress better for everyone.
